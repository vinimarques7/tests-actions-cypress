# tests-actions-cypress

 O test-actions-cypress é um repositório criado do zero como laboratório prático de aprendizado em DevOps. Ele não é um produto, mas sim uma jornada documentada em código — cada arquivo, pipeline e configuração representa um conceito aprendido e aplicado na prática.

 Este repositório surgiu dos estudos mentorados pela Inteligência Artificial, onde o CI/CD foi bastante relacionado. O desafio dado à mim na empresa foi de migrar pipelines de testes de Python para TypeScript e depois substituí-los pelo Cypress. A partir disso, o repositório evoluiu naturalmente para cobrir o ciclo completo de CI/CD — desde a criação dos primeiros testes até o build, versionamento e deploy automático de uma image Docker.

 A filosofia do projeto é simples: **aprender fazendo**. Cada erro resolvido, cada pipeline passando no GitHub Actions e cada conceito entendido está registrado na história de commits do repositório.

 Através deste repositório você vai passar por conceitos básicos de DevOps, explicações e também como eu utilizei a IA para me ajudar a maximizar meus estudos
